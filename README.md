# beabee-common

Shared code between Beabee projects for Node.js, Deno and the browser.

## Installation

```bash
npm install @beabee/beabee-common --save
```

## Usage

```typescript
import { parseDate } from "@beabee/beabee-common";

parseDate("2022"); // [ 2021-12-31T23:00:00.000Z, 'y' ]
```
